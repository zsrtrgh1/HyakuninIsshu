# HyakuninIsshu
百人一首を四択でランダムに出題するサイトです。<br>
ブラウザによっては、句を読み上げてくれます。<br>
出題範囲を指定できるので効率よく覚えることができます。
## 設定
出題については次の設定を行うことができます。

- 出題範囲
- 表示方法(かな/漢字)
- 濁点の消去(かな表記の場合)
